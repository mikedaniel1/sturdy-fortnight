Array

   Arrays are generally described as "list-like objects"; they are basically single objects that contain multiple values stored in a list. Array objects can be stored in variables and dealt with in much the same way as any other type of value, the difference being that we can access each value inside the list individually, and do super useful and efficient things with the list.
Array methods are:

  • Array join()

  • Array toString()

  • Array push()

  • Array pop()

  • Array at()

Array Join()

The join() method also joins all array elements into a string.

EXAMPLE for Join

const fruits = ["Banana", "Orange", "Apple", "Mango"]; document.getElementById("demo").innerHTML = fruits.join(" * ");

Array toString()

The JavaScript method toString() converts an array to a string of (comma separated) array values.
JavaScript automatically converts an array to a comma separated string when a primitive value is expected.

EXAMPLE For toString

const fruits = ["Banana", "Orange", "Apple", "Mango"]; document.getElementById("demo").innerHTML = fruits.join(" * ");

Array push()

The push() method adds a new element to an array (at the end)

EXAMPLE For push

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.push("Kiwi");

Array pop

The pop() method removes the last element from an array

EXAMPLE For pop

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.pop();

Array at

The at() method of Array instances takes an integer value and returns the item at that index, allowing for positive and negative integers. Negative integers count back from the last item in the array.

EXAMPLE For at

const array1 = [5, 12, 8, 130, 44];

let index = 2;

console.log(`An index of ${index} returns ${array1.at(index)}`);

index = -2;

console.log(`An index of ${index} returns ${array1.at(index)}`);
